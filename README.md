# Expense Tracker

A Flutter Expense Tracker application with Firebase Authentication and Cloud Firestore.

## Features

- User Login & Signup
- Add Income
- Add Expense
- Edit Transaction
- Delete Transaction
- Dashboard
- Balance Summary
- Pie Chart Analytics
- Search Transactions
- Category Filter
- PDF Export

## Technologies Used

- Flutter
- Firebase Authentication
- Cloud Firestore
- fl_chart
- pdf
- printing

## Installation

1. Clone this repository

```bash
git clone https://github.com/YOUR_USERNAME/expense_tracker.git
```

2. Install packages

```bash
flutter pub get
```

3. Configure Firebase

Run

```bash
flutterfire configure
```

4. Run

```bash
flutter run
```
